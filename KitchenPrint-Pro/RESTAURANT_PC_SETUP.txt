KitchenPrint-Pro - Restaurant PC Setup Guide
============================================================
Target PC: MCPOSE
OS: Windows Embedded Standard (32-bit, Windows 7 SP1)
CPU: Intel Celeron J1900 @ 1.99 GHz
RAM: 4 GB (2.89 GB usable)
============================================================


STEP 1 - Install Python 3.8.10 (32-bit)
----------------------------------------
Windows Embedded Standard is 32-bit. You MUST use the 32-bit installer.

Download link (32-bit):
  https://www.python.org/ftp/python/3.8.10/python-3.8.10.exe

During install:
  - Check "Add Python to PATH"  <-- REQUIRED
  - Check "Install for all users" (recommended)

Verify in a command prompt:
  python --version
  Should show: Python 3.8.10


STEP 2 - Copy the project to the PC
-------------------------------------
Copy the entire KitchenPrint-Pro folder to:
  C:\KitchenPrint-Pro

Or wherever you want. Just note the path.


STEP 3 - First Launch (auto-installs everything)
--------------------------------------------------
Double-click SmartLaunch.bat

On first run it will:
  1. Detect that packages are not installed
  2. Upgrade pip
  3. Install all required Python packages from requirements.txt
  4. Open firewall ports automatically
  5. Start the server

The window title will show [MCPOSE] to confirm the correct PC.

After the first run, a file "data\.installed" is created.
Future launches skip the install step and start immediately.

If you ever need to reinstall packages (after update), delete:
  data\.installed
Then run SmartLaunch.bat again.


STEP 4 - Install Tesseract OCR (for PDF order parsing)
--------------------------------------------------------
Required for reading order PDFs from UberEats / DoorDash AirPrint.

Download (Windows 32-bit compatible):
  https://github.com/UB-Mannheim/tesseract/wiki
  Download the 32-bit installer: tesseract-ocr-w32-setup-*.exe

Install to the default location:
  C:\Program Files (x86)\Tesseract-OCR

SmartLaunch.bat auto-detects this path. No extra config needed.


STEP 5 - Install Ghostscript (for image-based PDFs)
-----------------------------------------------------
Some AirPrint PDFs are image-based and cannot be read by text extraction alone.
Ghostscript renders them to images, which Tesseract then reads.

Download (32-bit Windows):
  https://github.com/ArtifexSoftware/ghostpdl-downloads/releases
  Download: gs*.w32.exe  (the 32-bit version)

Install to default location. SmartLaunch.bat auto-detects it.

Without Ghostscript, the PDF pipeline still works for text-based PDFs
using pypdf and pdfplumber (both installed automatically).


STEP 6 - Install your kitchen printers in Windows
---------------------------------------------------
Before configuring KitchenPrint-Pro, make sure the printers are
added in Windows (Control Panel -> Devices and Printers).

To list all installed printer names, open a command prompt and run:
  python -c "import win32print; print('\n'.join(str(p[2]) for p in win32print.EnumPrinters(2)))"

Copy the exact printer name (spelling and spacing matter).


STEP 7 - Configure printers in the web app
-------------------------------------------
After SmartLaunch.bat starts the server, open a browser on the PC:
  http://127.0.0.1:5000

1. Click the gear/settings icon
2. Set:
   - Kitchen Printer  = your kitchen printer (exact Windows name)
   - Sushi Printer    = your sushi printer
   - Packer Printer   = your full-order / original printer
3. Save settings


STEP 8 - Find the PC LAN IP address
-------------------------------------
Open a command prompt and run:
  ipconfig

Look for "IPv4 Address" under your active network adapter (Ethernet or Wi-Fi).
Example: 192.168.1.50

The web app is accessible from any device on the same network at:
  http://192.168.1.50:5000

Replace 192.168.1.50 with the actual IP of MCPOSE.


STEP 9 - Connect iPhone/iPad for AirPrint
------------------------------------------
1. iPhone/iPad must be on the SAME Wi-Fi / LAN as MCPOSE
2. Open Uber Eats / DoorDash / Grubhub app on the iPad
3. Open an order and tap Print
4. The printer "KitchenPrintPro" should appear
5. Select it and print
6. The order appears in Incoming Orders on the web UI
7. Tap Accept to print to the kitchen


STEP 10 - If AirPrint printer does not show on iPad
-----------------------------------------------------
The firewall ports should be opened automatically by SmartLaunch.bat.
If AirPrint still does not show, run these manually as Administrator:

  netsh advfirewall firewall add rule name="KitchenPrint IPP" dir=in action=allow protocol=TCP localport=8631
  netsh advfirewall firewall add rule name="KitchenPrint Web" dir=in action=allow protocol=TCP localport=5000
  netsh advfirewall firewall add rule name="KitchenPrint RAW9100" dir=in action=allow protocol=TCP localport=9100
  netsh advfirewall firewall add rule name="KitchenPrint mDNS" dir=in action=allow protocol=UDP localport=5353

Then:
  - Reconnect Wi-Fi on iPad
  - Restart SmartLaunch.bat
  - Try printing again

On Windows Embedded Standard, also run the legacy netsh command:
  netsh firewall add portopening TCP 8631 "KitchenPrint IPP" ENABLE ALL
  netsh firewall add portopening UDP 5353 "KitchenPrint mDNS" ENABLE ALL


STEP 11 - Verify everything is running
----------------------------------------
Check ports are open (in command prompt):
  netstat -ano | findstr ":5000"
  netstat -ano | findstr ":8631"

Both should show LISTENING.

Check web app:
  http://127.0.0.1:5000


PDF PARSING - How it works on MCPOSE (32-bit Win7)
====================================================
Order PDFs received via AirPrint go through 4 steps in order:

1. pypdf         - pure Python text extraction (fastest, works on all PDFs)
2. pdfplumber    - pure Python, better layout analysis (backup text extractor)
3. Ghostscript   - renders PDF pages to images, then Tesseract reads them
                   (requires Ghostscript 32-bit installed - see Step 5)
4. pymupdf/fitz  - last resort renderer (may not work on 32-bit Win7)

For MCPOSE (32-bit Win7), steps 1 and 2 handle most order PDFs.
Steps 3 and 4 are used only if the PDF is image-based (scanned).

TROUBLESHOOTING
================
Problem: "Python not found"
  -> Reinstall Python 3.8.10 32-bit and check "Add Python to PATH"

Problem: pip install fails for pymupdf
  -> OK to ignore on 32-bit Win7. PDF parsing still works via pypdf + pdfplumber.

Problem: PDFs arrive but orders show as UNPARSED
  -> Install Tesseract (Step 4) and Ghostscript (Step 5)
  -> Check data\print_jobs\*.txt to see what text was extracted

Problem: AirPrint not visible on iPad
  -> See Step 10 above (firewall ports)

Problem: Printer not printing
  -> Open http://127.0.0.1:5000 -> Settings -> verify printer names
  -> Run the python -c "win32print..." command from Step 6 to list exact names
